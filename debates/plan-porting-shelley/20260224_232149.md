## Critical Flaws

### 1. Codex history injection is O(n²) tokens

```diff
- const history: { role: "user"|"assistant"; content: string }[] = [];
- // Build prompt as flattened history + new message
+ // Each codex turn injects ALL prior turns. Round N sends N×avg_response tokens.
+ // 5-round debate: Agent B alone sends ~5× the raw content.
+ // Either cap history window or accept this cost explicitly in the design doc.
```

This isn't a stub concern — it's an architectural decision that affects cost estimates in the stats footer.

---

### 2. Log safety regression

Accumulating then overwriting is **worse** than `tee -a`:

```diff
- const flushLog = () => Bun.write(logFile, logBuffer); // overwrite
+ // SIGKILL, OOM, or Bun crash → zero log output
+ // Shell's tee -a wrote each line immediately
```

Fix: append per `log()` call, or at minimum flush after every agent response:

```typescript
const log = (line: string) => {
  logBuffer += line + "\n";
  Bun.write(logFile, logBuffer);  // small cost, high safety
  console.log(line);
};
```

---

Not ambiguous — shelley.sh already proves `--session-id $OUR_UUID` + `--resume $OUR_UUID` works in production. We own the UUID. Proposed revision was wrong to suggest reading it from the response.

Revised callClaude sketch:

```typescript
async function callClaude(opts: CallOpts): Promise<LLMResult> {
  const args = ["claude", "-p", "--output-format", "json", "--model", opts.model];
  if (opts.resume) {
    args.push("--resume", opts.resume);
  } else {
    if (opts.systemPrompt) args.push("--system-prompt", opts.systemPrompt);
    args.push("--session-id", opts.sessionId!);  // caller owns this UUID
  }
  args.push(opts.prompt);

  const env = { ...process.env };
  delete env.CLAUDE_CODE;
  const proc = Bun.spawn(args, { env, stdout: "pipe", stderr: "pipe" });
  const [out, err] = await Promise.all([
    new Response(proc.stdout).text(),
    new Response(proc.stderr).text(),
  ]);
  await proc.exited;
  if (err) appendFileSync(logFile, `<!-- stderr: ${err.trim()} -->\n`);

  const json: ClaudeJsonOutput = JSON.parse(out);
  if (json.is_error) throw new Error(`claude error: ${json.result}`);
  return {
    text: json.result,
    inputTokens: json.usage.input_tokens,
    outputTokens: json.usage.output_tokens,
    costUsd: json.cost_usd,
    durationMs: json.duration_ms,
  };
}
```
