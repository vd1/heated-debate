- Add a small `TOOL_DIGEST_DRIFT_ALLOWLIST` map: plugin-name → project-slug. Drift on an allowlisted plugin is auto-eligible for topic selection regardless of version-bump class. This is what makes drift useful at low volume — one well-curated allowlist beats a noisy global signal.
