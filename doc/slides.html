Decision · Runtime Mode

Choose the path that keeps cold-start p90 ≤12m

Option A — Subscription CLI

shelley.ts · bun

  • No API key needed — subscription login only
  • claude or codex backends interchangeable
  • ~50s cold-start per turn; higher latency p90
  • Env sanitation strips keys before each spawn
  • Simpler auth; works today
Option B — API Orchestration

Agent SDK · persistent process

  • Requires ANTHROPIC_API_KEY — API billing
  • Persistent in-process agents, no per-turn spawn
  • Native Python event loop; token-accurate logging
  • Eliminates cold-start overhead
  • Enable once debate protocol is stable

Use Option A for trials; graduate to Option B when cold-start p90 exceeds gate or token visibility matters.

Decision · Control Mode

Escalate to tournament only on high-risk changes

Default — Single Debate

One Architect vs One Reviewer

  • Normal PRs, feature specs, refactors
  • 5 rounds · dial 5→1
  • Single timestamped transcript
  • Human makes final decision
Escalation — Tournament + Judge

N parallel debates · scored

  • Correctness, security, data-loss, compliance changes
  • Different dial vectors or model pairings
  • Judge agent (or 3-panel) scores outputs
  • Winner advances to next phase

Escalation trigger: change touches correctness, security, compliance, or irreversible data operations.

Operating Policy

Clear rules, explicit owner, defined exit

✓ Use when
  • Ambiguous, high-impact design decisions
  • Competing implementation approaches
  • High-risk PRs with security / data exposure
  • Latency budget allows cold-start p90 ≤12m
✗ Do not use when
  • Trivial refactors or formatting changes
  • Urgent hotfixes requiring <5m response
  • Low-risk edits with obvious single solution
  • Token budget already exceeded for the sprint
Owner
  • Human makes the final architectural call
  • Transcript is the audit trail, not the decision
  • Reviewer of record signs off before merge
Exit criteria
  • Unresolved critical defects = zero
  • All open questions answered in transcript
  • Plan has no contradictions across rounds
